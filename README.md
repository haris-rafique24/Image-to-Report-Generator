To run the frontend of the application, change the directory to image-report-app/frontend. Then execute the following commands:
```
npm install
npm run dev
```

To run the backend of the application, change the directory to image-report-app/backend. Then, execute the following command:
```
uvicorn main:app --reload --port 8000
```


Make sure to get your own API key from https://aistudio.google.com/. Happy learning!
